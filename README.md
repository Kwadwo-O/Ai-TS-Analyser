Brief;

  Welcome to our AI Text Speech Analyser,

  this program uses AI via openrouter to generate a short, medium or long sentence based on difficulty and the prompt request and requires the user to type that sentence as quick as they can with minimal error. The completed sentence is then outputted in a seperate prompt to AI and the user is given a score based on     their performance. 

  All responses are given a set score by AI based on how similar they are to the original sentence. 

  The program has three different gamemodes; Normal (Basic sentence), Code and memory game.

Normal;

  This gamemode requires the user to write a basic sentence as fast as they can with minimal error. 

Code;

  This gamemode requires the user to write a section of code generated by the AI.

Memory game;

  This gamemode requires the user to remember a basic sentence and when the sentence dissapears they have as much time as they need to try write it as close as they can to the original sentence.
